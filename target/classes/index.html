Sobre o Projeto

A Locadora de Filmes é um sistema desenvolvido para facilitar a administração de um catálogo de filmes, funcionários e usuários de uma locadora de filmes. Seu principal objetivo é oferecer uma plataforma intuitiva e eficiente para a gestão de todas as operações relacionadas à locação e devolução de filmes.

Temos como objetivo fornecer uma interface amigável para os usuários realizarem suas locações e consultarem informações sobre os filmes disponíveis.
As funcionalidades principais incluem cadastro de filmes, cadastro de funcionários e usuário e locação e devolução de filmes.
